# FRONT-OUBREAK-GAME

## Deploy frontend (Vercel)

No Vercel, configure a variavel de ambiente:

`VITE_SOCKET_URL=https://seu-backend.onrender.com`

Sem essa variavel, o modo online nao conecta em producao.

## Desenvolvimento local

Opcionalmente, em dev ele usa `http://localhost:3000` por padrao.
